# Portfolio Website

A modern and responsive personal portfolio website built using **React.js** to showcase my skills, projects, certifications, achievements, and professional journey.


---

## 📌 About the Project

This portfolio website serves as my personal digital portfolio, highlighting my technical skills, academic background, certifications, achievements, and projects. The website is designed to provide visitors with a clean and professional overview of my work and experience.

---

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Professional landing page
* About Me section
* Skills showcase
* Projects portfolio
* Certifications section
* Achievements section
* Contact information
* Smooth user experience with modern UI

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/leepakrishna-crypto/Portfolio.git
```

### Navigate to the project directory

```bash
cd Portfolio
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
